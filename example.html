Conway's Game of Life - JavaScript Package Demo

Basic Usage

This demonstrates the simplest way to use the package:

GameOfLife.create('game-container');

Custom Configuration

Example with custom settings:

GameOfLife.create('game-custom', {
    width: 600,
    height: 400,
    cellSize: 8,
    speed: 150,
    initialPattern: 'glider',
    fillColor: '#ff6b6b',
    backgroundColor: '#f8f9fa'
});
